# 书籍资源导航站

一个纯前端的书籍导航网站模板，专门用于收录电子书资源、阅读平台和书籍工具。

## 已实现功能

- 分类导航（按你指定的 8 个主分类）
- 左侧目录（快速跳转 + 滚动自动高亮 + 可收起成图标栏）
- 科技感视觉（高光卡片与动效层次）
- 全局搜索（站点名 / 简介 / 标签 / 分类）
- 分类筛选与结果统计
- 深浅色主题切换
- 响应式布局（手机和桌面都可用）

## 本地运行

直接双击打开 `index.html` 即可使用。

如果你希望走本地服务：

```bash
npx serve .
```

## 如何自定义

- 修改内置资源：编辑 `script.js` 中的 `BUILTIN_SITES`
- 修改分类顺序：编辑 `script.js` 中的 `CATEGORY_ORDER`
- 修改分类图标：编辑 `script.js` 中的 `CATEGORY_ICON`
- 调整视觉风格：编辑 `styles.css` 中 `:root` 的颜色变量
